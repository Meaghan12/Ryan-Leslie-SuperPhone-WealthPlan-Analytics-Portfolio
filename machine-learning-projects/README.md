# Machine Learning Projects

This folder contains applied machine learning projects demonstrating predictive modelling, classification, model evaluation, and business interpretation.

## Project Areas

- Churn risk and retention thresholds
- Content popularity prediction
- Loess model validation
- Classification and prediction workflows
- Feature importance and model interpretation

## Relevance to Ryan Leslie / SuperPhone

These projects show how machine learning can support:

- Audience segmentation
- Fan retention strategy
- Campaign targeting
- Creator monetization
- Customer behaviour prediction
- Strategic decision-making

The focus is not only on building models, but on translating model results into practical business recommendations.
