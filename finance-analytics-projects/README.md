# Finance Analytics Projects

This folder contains applied finance analytics projects focused on market data analysis, return distributions, volatility, forecasting, and investment interpretation. The work demonstrates how statistical diagnostics and time-series methods can turn raw financial data into practical insight for business, investment, and creator-economy strategy.

## Included Projects

### 1. Return Distribution Analysis: Microsoft & Apple

This project examines the distributional behaviour of Microsoft monthly prices and returns, along with Apple daily price and return movements. The analysis applies core financial analytics techniques including outlier assessment, Winsorization, descriptive statistics, distribution fitting, and visual return analysis.

The project evaluates Microsoft prices for skewness and outlier influence, compares Winsorization choices, and argues that returns are generally more useful than price levels for risk and forecasting analysis. Apple’s January 2020 price and return behaviour is also used to demonstrate how time-series plots and return histograms can support interpretation of market movement, volatility, and financial behaviour.

**Skills demonstrated:**

- Financial return analysis  
- Outlier detection and Winsorization  
- Distribution fitting  
- Lognormal and Normal model interpretation  
- Descriptive statistics  
- Visual financial data analysis  
- Market-risk interpretation  

**Portfolio relevance:**  
This project shows the foundation of financial analytics: understanding how asset prices and returns behave before building forecasts, simulations, or decision models.

### 2. Bitcoin Price and Volatility Forecasting

This project analyzes Bitcoin price, return, volume, and volatility behaviour using forecasting and time-series diagnostics. The analysis compares moving-average and exponential-smoothing forecasts, examines ACF and PACF patterns, fits an AR(1) model to log returns, and investigates volatility clustering using squared returns.

The project finds that Bitcoin prices are highly non-stationary, while daily log returns behave closer to white noise in the conditional mean. However, squared returns show evidence of volatility persistence, supporting the use of ARCH/GARCH-type modelling for future volatility analysis.

**Skills demonstrated:**

- Time-series forecasting  
- Moving-average forecasting  
- Simple exponential smoothing  
- ACF and PACF interpretation  
- AR(1) modelling  
- Stationarity assessment  
- Volatility clustering analysis  
- ARCH/GARCH reasoning  
- Financial risk interpretation  

**Portfolio relevance:**  
This project demonstrates the ability to move beyond surface-level price charts and evaluate whether financial data contains forecastable structure, volatility persistence, and risk signals.
